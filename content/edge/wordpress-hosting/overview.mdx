## What this covers

- Attaching a custom domain and pointing DNS records correctly
- Changing WordPress defines such as `WP_DEBUG`
- Browsing persistent files in the `wp-content` volume
- Running WP-CLI commands over SSH
- Opening the database in the built-in DB Explorer
- Understanding why the Wasmer plugin must remain installed
