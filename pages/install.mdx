# Install Wasmer

import { Callout } from "nextra-theme-docs";
import { Tabs, Tab } from "nextra-theme-docs";

Wasmer allows you to run applications either **Standalone** (_via the CLI_) or **Embedded** within [other languages](./#sdks) such as C/C++, Rust, Python, Go, PHP, Ruby...

<Tabs items={['CLI (Bash)', 'CLI (PowerShell)', 'JS' /*, 'Python' */]}>
  <Tab>
    **Unix** (macOS, Linux):

    ```bash copy
    curl https://get.wasmer.io -sSfL | sh
    ```

    Or, to install a specific version:

    ```bash /4\.3\.0/ copy 
    curl https://get.wasmer.io -sSfL | sh -s "v4.3.0"
    ```

    <Callout type="info">
      Other installation methods are available here:
      https://github.com/wasmerio/wasmer-install
    </Callout>


    ### Is Wasmer up to date?
    Wasmer updates very frequently. If you have installed Wasmer some time ago, chances are your Wasmer version is out of date. Get the latest version of Wasmer by running:

    ```bash copy
    wasmer self-update
    ```


  </Tab>

  <Tab>
    **PowerShell** (Windows):

    ```bash copy
    iwr https://win.wasmer.io -useb | iex
    ```

    Or, to install a specific version:

    ```bash /4\.3\.0/ copy
    $v="v4.3.0"; iwr https://win.wasmer.io -useb | iex
    ```

    <Callout type="info">
      Other installation methods are available here:
      https://github.com/wasmerio/wasmer-install
    </Callout>

    ### Is Wasmer up to date?
    Wasmer updates very frequently. If you have installed Wasmer some time ago, chances are your Wasmer version is out of date. Get the latest version of Wasmer by running:

    ```bash copy
    wasmer self-update
    ```


  </Tab>


  <Tab>
    **NPM**:

    ```bash copy
    npm install @wasmer/sdk --save
    ```

    Bun:

    ```bash /@wasmer/ /sdk/ copy
    bun install @wasmer/sdk --save
    ```
    
    Read the [Wasmer-JS SDK docs](/sdk/wasmer-js).
  </Tab>
