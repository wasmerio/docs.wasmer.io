[Wasmer](https://wasmer.io/) is a container system based on WebAssembly that allows running applications anywhere: in the browser, embedded in your favorite programming language, or standalone in the server.

> Wasmer mission is make all software universally available

## Ecosystem


1. [Wasmer Runtime](/runtime): allows running Wasmer containers anywhere
1. [Wasmer Registry](/registry): explore containers from other users and to publish and distribute your own ones.
2. [Wasmer Deploy](/deploy): run containers on Wasmer distributed Edge
